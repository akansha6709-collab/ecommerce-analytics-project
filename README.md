# E-Commerce Analytics Project

## 📊 Overview
This project analyzes customer behavior and revenue distribution in an e-commerce dataset using Python, SQL, and Power BI.

The goal is to identify key revenue drivers and provide actionable business insights.

---

## 🛠️ Tools Used
- Python (Data Cleaning & Feature Engineering)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)

---

## 📂 Dataset
- Customer demographics
- Purchase behavior
- Product categories
- Transaction-level data

---

## 🔍 Key Analysis

### 1. Customer Segmentation
- Customers divided into High-value and Low-value groups
- High-value customers contribute a significantly larger share of revenue

### 2. Category Performance
- Clothing contributes ~45% of total revenue
- Accessories contribute ~30%+
- Other categories contribute less comparatively

### 3. Customer Insights
- Revenue is heavily dependent on high-value customers
- Age does not significantly impact customer value
